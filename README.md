# HelperFinder — сервис для поиска помощников и наставников

## Суть проекта
HelperFinder — это прототип приложения, которое помогает людям находить живых помощников для учебных задач, консультаций или бытовых дел.

Пользователь выбирает, какая помощь ему нужна -> ему показываются люди, которые могут помочь -> он связывается напрямую.

## Какая проблема решается?
Когда человеку срочно нужна помощь — он не знает, к кому обратиться.  
Поиск по интернету занимает много времени, нет отзывов, нет удобного выбора.

## Наше решение
Собрать помощников в одном месте.  
Дать пользователю возможность:
>выбрать тип помощи
>посмотреть список людей
>связаться с любым напрямую

## MVP (прототип)
Прототип реализован в виде Notion-страницы:
> список помощников
>навыки и описания
